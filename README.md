# FinTrack Pro – Personal Finance & Expense Management System

## About the Project

FinTrack Pro is a Personal Finance and Expense Management System designed to help users manage their daily finances in a simple, organized, and visual way.

Many people struggle to track their income, expenses, budgets, and savings goals. This project was built to provide a single platform where users can monitor their financial activities, understand their spending habits, and make better financial decisions.

The application offers an interactive dashboard, budget tracking, financial goal management, spending analytics, notifications, and data management features. All user data is stored locally in the browser using Local Storage, allowing the application to work without a backend or database.

This project was developed as a frontend-focused application to strengthen practical skills in JavaScript, DOM Manipulation, Local Storage, Responsive Web Design, and Data Visualization.
